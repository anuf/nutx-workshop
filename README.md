# nutx-workshop

A taste on __nuxt__ with __vue__ to create Universal applications.

[https://codigofacilito.com/cursos/taller-nuxt-vue](https://codigofacilito.com/cursos/taller-nuxt-vue)

## Installation
npm install nuxt axios vue-router bulma


## Execution
npm run dev
